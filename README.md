# robotlab-x
Core Framework

Modern robotics framework with a focus on visualization

## Development

### Prerequisites

- Node.js v21.7.2 or higher
- Yarn v1.22.19 or higher

### Clone the repository

```bash
git clone https://github.com/RobotLab-X/robotlab-x.git
cd robotlab-x
```

### Install dependencies

```bash
yarn install-all
```

### Run the application

```bash
yarn start-dev
```
